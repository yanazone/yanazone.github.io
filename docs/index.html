Quick Questions and Answers

What is Hostmann's Stratetegic Table?
Its a spiritually independent from anything you have played game that takes inspiration from the roots: real life. During my long and frustrating attempts to create a fun experience with HoI4 lobbies and mods I've identified core design issues that limit the social aspects and mechanical flaw of Paradox titles. Using that as a foundation I want to rethink strategy genre and create both fun and realistic, deep and straightworward game that also does not rip players off with endless DLCs and has high quality assurance.
Where can I try the game?
Currenty we are in a alpha phase of development. In order to check the game out you need to join Discord and wait for the next (usually bi-weekly) alpha build. Simpy tag @Hostman and I will get in touch!
Will it be published on Steam? Will it be free?
Eventual release will see cross-platform play through Steam, Apple Store and Google Play. (You hear that rigth - mobile version is in the works!). During the closed test the game is free but upon open beta release the game will be paid. This is because its not an open source project and I spend real effort and resources to develop it. Projected price will be way below large industry titles.
Do you use Ai in development?
This is a passion project and I use minimal if any AI in development, due to ethical principle and because I enjoy developing! While this project is not a democracy where anyone can suggest features I still keep the soul of a human/community-first enviroment.
